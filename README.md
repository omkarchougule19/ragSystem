# 🚀 RAG Chatbot using LangChain, OpenAI & ChromaDB

An end-to-end Retrieval-Augmented Generation (RAG) chatbot project built using Python, LangChain, OpenAI Embeddings, and ChromaDB.

This project demonstrates how modern AI applications retrieve relevant information from documents and generate intelligent responses using Large Language Models (LLMs).

---

# 📌 Features

- ✅ Document ingestion pipeline
- ✅ PDF/Text document processing
- ✅ Semantic chunking & preprocessing
- ✅ OpenAI embeddings integration
- ✅ ChromaDB vector database
- ✅ Similarity search retriever
- ✅ Conversational RAG pipeline
- ✅ LangChain orchestration
- ✅ Jupyter Notebook workflow
- ✅ Modular and extensible architecture
